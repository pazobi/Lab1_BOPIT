Para este proyecto se buscó trabajar en una versión de la aplicación 
del laboratorio anterior, en donde tuvimos que crear una plataforma
simulando el juego "Bop It" para teléfono claro que sin implementar el
videojuego en si, sino que enfocarse en la interaccioón del usuario con
la app, la creación de distintas pantallas y lo que contienen.
En esta nueva vesión de la aplicación se implementó la reproducción de
música en la aplicación según el contexto del problema. Los requisitos 
fueron los siguientes:
-Uso de MediaPlayer con múltiples "pistas" de audio.
-Se debe establecer una para la "Victoria".
-Se debe establecer una para la "Derrota".
-Se debe establecer una como música de fondo.
-Uso del GestureDetector
